    OPTIMIZATION CONSIDERATIONS:

  • Tile layers support fast visualization of large data sets
  • Feature layers are more flexible and interactive than tile layers
    but slower drawing times due to frequent app requests to source data

    • tip: for more than 4,000 data point features or 2,000 line/polygon features, or size over 10 MB, publish maps as hosted feature layer
    TYPES OF MAP SERVICES

  • WMS (Web Mapping Service): similar to ArcGIS, generates defined bounds & features image, processing intensive
  • WFS (Web Feature Service): provides the vector features to the browser to display, selectable & editable
